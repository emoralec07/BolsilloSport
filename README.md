BolsilloSport 💰⚽

App para gestionar el bankroll de apostadores deportivos. Lleva el control de tus apuestas como un profesional.

¿Qué hace?


Establece tu bankroll inicial

Registra apuestas con deporte, descripción, monto y cuota

Marca apuestas como ganadas, perdidas o pendientes

Resuelve apuestas pendientes cuando ya tienes el resultado

Muestra estadísticas en tiempo real:


Bankroll actual

Ganancia o pérdida total

Porcentaje de acierto

Total de apuestas registradas
